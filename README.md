# MScript-binary-ESP32S3-DevKitC1版


MicomScript は、C 言語に似た構文を持つマイコン制御用スクリプト言語です。
逐次コンパイル不要で、即時実行が可能です。


## Features
- Cライクな制御文
- 即時実行（インタプリタ方式）
- ESP32S3に最適化


## Quick Start
1. Arduino IDE をインストール  
2. ESP32 を接続  
3. MScript.ino を開く  
4. 必要ライブラリをインストール  
5. コンパイルして書き込み

**詳細なセットアップ手順はこちら↓**
-[doc/setup_esp32/setup_esp32.md](doc/setup_esp32/setup_esp32.md)
