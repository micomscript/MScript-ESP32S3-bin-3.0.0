MicomScript Bluetooth API Specification

Bluetooth 通信仕様

1. Bluetooth 開始

Bluetooth.begin(name, mode)

引数1：デバイス名  
引数2：動作モード  
　true  = クライアント  
　false = サーバー（初期値）

クライアント例
Bluetooth.begin("client", true)  
"Hello!"

サーバー例
Bluetooth.begin("server")  
("server", false) と同じ  
"Hello!"



2. MAC Address の取得（サーバー側）

Bluetooth.getAddress()  
戻り値例：  
"AA:BB:CC:DD:EE:FF"



3. Bluetooth 接続（クライアント側）

Bluetooth.connect("AA:BB:CC:DD:EE:FF")

接続成功：  
Connected to Server!  
true

接続失敗：  
Failed to connect.  
false



4. 接続状態の確認

Bluetooth.isConnected()

接続中：true  
未接続：false



5. 文字列送信（サーバー側）

Bluetooth.write("Word\n")  
"OK"

※ 注意  
・末尾に '\n' が無い場合、自動的に付加される  
・ただし、基本的には明示的に '\n' を付けて送信することを推奨



6. データの有無確認

Bluetooth.available()  
データがある場合：1以上  
データが無い場合：0



7. 文字列受信（クライアント側）

Bluetooth.read()  
戻り値例：  
"Word"



8. Bluetooth 接続断

Bluetooth.disconnect()  
"OK"



9. Bluetooth 終了

Bluetooth.end()  
"See You!"
